Interface Facet<T>

  • Type Parameters:
    T - Type of items returned for this facet: TermFacetItem, RangeFacetItem.

    All Known Implementing Classes:
    AttributeTermFacetBase, FacetBase, Facets.Categories.Terms, Facets.DateTimeAttribute.Ranges, Facets.DateTimeAttribute.Terms, Facets.MoneyAttribute.Ranges, Facets.MoneyAttribute.Terms, Facets.NumberAttribute.Ranges, Facets.NumberAttribute.Terms, Facets.Price.Ranges, Facets.StringAttribute.Terms

    public interface Facet<T>
    Facet 'component' that supports keeping the state of selected items in application's query string. See FacetParser for reconstructing state of multiple facets from application's URL. Use {#getUrlParams} to construct URLs for selecting individual facet items. You can also use helper methods {#getSelectLink}, {#getUnselectLink} and {#isSelected} that make use of {#getUrlParams}.
