      • getId

        public java.lang.String getId()
        The id of the address, assigned by the Sphere backend. The id is represents a snapshot of a customer's address: each change of customer's address (using CustomerService.update(io.sphere.client.model.VersionedId, io.sphere.client.shop.model.CustomerUpdate)) means creating a new address with a new id and replacing the old address with it.
