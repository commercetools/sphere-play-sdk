      • createPasswordResetToken

        public CustomerToken createPasswordResetToken(java.lang.String email)
        Description copied from interface: CustomerService
        Creates a password reset token for a customer. The validity of the token is 10 minutes.

        The typical workflow is the following:

        1. Customer enters his or her email in a password reset form and submits.
        2. Create a password reset token using this method.
        3. Send an email containing a link with the token to the customer.
        4. The link points to a form where the customer can enter a new password. The form should load the customer using byToken and store customer's id and version in hidden form fields.
        5. When the customer submits the form with the new password, call resetPassword or resetPasswordAsync passing in the new password and the token
        Specified by:
        createPasswordResetToken in interface CustomerService
        Parameters:
        email - Email address for which the token should be created.
