Interface SearchRequest<T>

  • All Known Implementing Classes:
    SearchRequestAdapter

    public interface SearchRequest<T>
    Request that uses a Sphere search API to fetch objects satisfying some conditions (including fulltext search) and provide faceting functionality. Search is currently only supported for products.
